In many practical applications there are,
∙ some easy-to-measure quantities (generally called Xs)
− Age; Gender; Income; Education level; etc.

∙ and a difficult-to-measure quantity (generally called the Y)
− Amount of loan to give; Will she buy or not; How many days will he stay in the
hospital; etc.

Supervised learning is about computing the Y using the
Xs, assuming availability of data samples with Xs and
corresponding Ys (usually historical data)

Unsupervised learning is about computing patterns
within easy to measure attributes (the Xs)
